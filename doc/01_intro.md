## Stack

- `GHC` is the Haskell compiler
- `ghci` is the "GHC Interactive"
  - REPL (Read Eval Print Loop)
  - Similar to `python`, `ruby`, `node`
- `stack`
  - Downloads `ghc`
  - Resolves package dependencies
